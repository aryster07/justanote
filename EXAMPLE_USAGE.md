# Example: How to Use ChatGPT Graphics Prompts

This is a practical example showing how to use the prompts in `CHATGPT_GRAPHICS_PROMPTS.md` to create graphics for Just A Note.

## üìù Example Scenario

**Goal:** Create an Instagram post showcasing the music feature of Just A Note.

## üöÄ Step-by-Step Process

### Step 1: Choose Your Prompt
Open `CHATGPT_GRAPHICS_PROMPTS.md` and find "Music Feature Showcase" (Prompt #3):

```
Create an Instagram post (1080x1080px) for "Just A Note" highlighting the music 
feature:
- Split design: left side shows a stylized music player with golden play button
- Right side has text: "Add Your Favorite Song üéµ" in elegant font
- Background: Soft white to light gold gradient
- Floating musical notes around the player
- Bottom text: "Make your note unforgettable with music"
- Brand colors: gold #C5A059, white, and soft grays
- Modern, clean, romantic style
- Include small sparkle accents
```

### Step 2: Open ChatGPT
1. Go to https://chat.openai.com
2. Make sure you have access to DALL-E (GPT-4 with image generation)
3. Start a new chat

### Step 3: Paste the Prompt
Copy the entire prompt and paste it into ChatGPT exactly as written.

### Step 4: Review the Generated Image
ChatGPT will generate an image based on your prompt. Review it and check:
- ‚úÖ Is the size correct? (1080x1080px)
- ‚úÖ Are the colors right? (Gold #C5A059)
- ‚úÖ Does it match the Just A Note brand?
- ‚úÖ Is the text readable and well-placed?
- ‚úÖ Does it have the right elements? (music player, text, sparkles)

### Step 5: Request Modifications (if needed)
If something needs adjustment, ask ChatGPT to modify:

**Example modification requests:**
```
"Make the gold color more vibrant and closer to #C5A059"
```

```
"Add more sparkles around the music player"
```

```
"Make the text larger and use a more elegant serif font"
```

```
"Create a variation with the music player on the right side instead"
```

```
"Add the Just A Note logo in the top-left corner"
```

### Step 6: Generate Variations
Ask for multiple versions to choose from:

```
"Create 3 variations of this design with different layouts"
```

### Step 7: Download and Use
1. Click on the image to view full size
2. Right-click and "Save Image As..."
3. Use the image in your Instagram post!

## üí° Pro Tips

### Tip 1: Chain Prompts for Complex Designs
Start with the base prompt, then add refinements:

```
First message: [Paste the original prompt]
Second message: "Add the website URL 'justanote.me' at the bottom"
Third message: "Make the background gradient more subtle"
```

### Tip 2: Reference the PROJECT_BRIEF.md
If ChatGPT's colors don't match exactly, reference the brief:

```
"Please use the exact gold color from the Just A Note brand: #C5A059. 
This is described in the PROJECT_BRIEF.md as 'Royal Gold' and should 
be used for all accent elements."
```

### Tip 3: Ask for Brand-Consistent Elements
```
"Add decorative elements that match the Just A Note brand: 
small gold sparkles, heart icons, and musical notes in a minimalist style"
```

### Tip 4: Request High Resolution
```
"Please generate this in the highest resolution possible suitable for 
Instagram posting (1080x1080px at 72 DPI minimum)"
```

### Tip 5: Save Successful Prompts
If a prompt works really well, save it! You can reuse it for similar graphics.

## ‚úÖ Quality Checklist

Before publishing any generated graphic, verify:

- [ ] Dimensions are correct for the platform
- [ ] Brand colors match (#C5A059 gold)
- [ ] Typography is legible and elegant
- [ ] Design is minimalist and clean
- [ ] Messaging aligns with Just A Note's tone (warm, heartfelt)
- [ ] All text is spelled correctly
- [ ] Image quality is high (no pixelation)
- [ ] Design works on both light and dark backgrounds
- [ ] Call-to-action is clear (if applicable)
- [ ] Logo/branding is present (if applicable)
